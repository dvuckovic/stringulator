Quick & dirty circular string art simulator.

[Demo]

## Project setup
```
npm install --no-save
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Runs tests
```
npm test
```
